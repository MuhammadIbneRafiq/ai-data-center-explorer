# AI Data Center Explorer
 
## Getting Started
 
1. Install dependencies:
 
    ```bash
    npm install
    ```
 
2. Start the development server:
 
    ```bash
    npm run dev
    ```
 
3. Open the URL shown in the terminal (by default `http://localhost:5173`).
 
## Notes
 
- This project is a React + Vite app.
